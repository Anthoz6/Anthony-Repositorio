RNI Frontend

Desarrollo del frontend para la Plataforma de Relaciones Internacionales de la Corporación Universitaria Rafael Núñez

En Desarrollo
Next.js TypeScript CSS3 Tailwind CSS

Descripción del Proyecto

RNI Frontend es la interfaz de usuario de la plataforma de Relaciones Internacionales, diseñada para facilitar la gestión de convenios, movilidades académicas, y procesos internacionales de la universidad. El desarrollo se enfoca en crear una experiencia intuitiva y moderna para estudiantes, docentes y personal administrativo.

Este proyecto representa mi contribución al desarrollo del lado del cliente, implementando interfaces responsivas y funcionales que se comunican eficientemente con el backend desarrollado en Spring Boot.

Hitos del Desarrollo

Fase 1: Diseño y Arquitectura

Definición de la estructura del proyecto, configuración de Next.js con TypeScript, y diseño de componentes base.

Fase 2: Autenticación

Implementación del sistema de login y registro.

Fase 3: Módulos Principales

Desarrollo de los módulos de Gestion de convenios, Gestion de actividades y Gestión de organizaciones.

Fase 4: Integración con Backend

Conexión con la API REST del backend, manejo de estados y actualización en tiempo real.

Fase 5: Testing y Optimización (En Progreso)

Pruebas de usabilidad, optimización de rendimiento y corrección de bugs.

Stack Técnico

Frontend Framework

  • Next.js con TypeScript
  • React Router DOM para navegación

Estilos y UI

  • Tailwind CSS para diseño utility-first
  • CSS Modules para estilos encapsulados
  • Font Awesome para iconografía
  • Animaciones CSS3 personalizadas
  • Design System consistente

Características Técnicas

  • Componentes reutilizables y modulares
  • Lazy loading para optimización de carga
  • Manejo de errores centralizado
  • Interceptores de Axios para autenticación
  • Validaciones del lado del cliente
  • Responsive design mobile-first
